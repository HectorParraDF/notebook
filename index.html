Ideal Gas in the Grand Canonical Ensemble

Since the particles of an ideal gas are nonlocalized, the partition function for each one-particle subsystem is directly proportional to its volume:

$$Q_1(V,T) = Vf(T),$$

where $f$ is some function of temperature only. Computing the grand partition function yields

$$\mathcal{Q}(z,V,T) = e^{zVf(T)}.$$

The thermodynamical quantities are:

$$P = zkTf(T), $$
 
$$N = zVf(T),$$

$$U = zVkT^2 f'(T),$$

$$A = NkT\ln z - zVkTf(T),$$

$$S = -Nk\ln z + zVk(Tf'(T) + f(T)).$$

The equation of state and the energy in terms of $N$ and $T$ are

$$PV = NkT,$$ $$U = NkT^2 f'(T)/f(T).$$

The specific heat is then

$$C_V = Nk\frac{2Tf(T)f'(T) + T^2 (f(T)f''(T) - (f'(T))^2)}{(f(T))2}$$

The Polyakov Action

The Polyakov action is

$$S = - \frac{1}{4\pi\alpha'}\int d^2 \sigma \sqrt{-g} g^{\alpha \beta } \partial_\alpha X^\mu \partial_\beta X^\nu \eta_{\mu \nu},$$

where the field $g_{\alpha \beta}$ is a dynamical metric on the worldsheet. We can interpret this as a bunch of scalar fields $X$ coupled to 2d gravity.

The equations of motion for $X^\mu$ are

$$\partial_\alpha (\sqrt{-g} g^{\alpha \beta}\partial_{\beta}X^\mu) = 0,$$

while the equations of motion for $g_{\alpha \beta}$ yield

$$g_{\alpha \beta} = 2f(\sigma)\partial_\alpha X \cdot \partial_\beta X,$$

where $f(\sigma)$ is defined as

$$f^{-1}(\sigma) = g^{\rho \sigma}\partial_\rho X \cdot \partial_\sigma X.$$

This action has the following symmetries:


1. $D$-dimensional Poincar√© invariance:

$$X^\mu \to \Lambda^\mu_\nu X^\nu + a^\mu,$$ $$g_{\alpha \beta} \to g_{\alpha \beta}.$$

2. Diffeomorphism invariance:

$$X^\mu (\sigma) \to \tilde X^\mu (\tilde \sigma) = X^\mu (\sigma),$$ $$g_{\alpha \beta}(\sigma) \to \tilde g_{\alpha \beta} (\tilde \sigma) = \frac{\partial \sigma^\gamma}{\partial \tilde \sigma^\alpha} \frac{\partial \sigma^\delta}{\partial \tilde \sigma^\beta} g_{\gamma \delta}(\sigma).$$

3. Two-dimensional Weyl invariance:

$$X^\mu (\sigma) \to X^\mu (\sigma),$$ $$g_{\alpha \beta}(\sigma) \to e^{2\phi (\sigma)}g_{\alpha \beta}(\sigma).$$

We can make the metric conformally flat (by reparameterization):

$$g_{\alpha \beta} = e^{2\phi(\sigma)}\eta_{\alpha \beta},$$

Using the Weyl invariance we can set $\phi(\sigma) = 0$, yielding a flat metric:

$$g_{\alpha \beta} = \eta_{\alpha \beta}.$$

The Polyakov action then takes the form

$$S = - \frac{1}{4\pi\alpha'}\int d^2 \sigma \partial_\alpha X \cdot \partial^\alpha X.$$

This reduces the equations of motion for $X$ to a wave equation with constraints

$$\dot X \cdot X' = 0, ~~~~~~~~~~~\frac{1}{2} ( \dot X^2 + X'^2 ) = 0.$$
